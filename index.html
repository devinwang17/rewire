《神經可塑性》：今天開始打破負面循環

先寫一下我的小註解：雖然這本書很多概念，可能早見於各種書籍文章，但重新從神經科學理解一次還是挺有趣的。很多時候，光是理解為什麼發生，似乎就能更釋然一點，也更知道如何鬆動既有的信念和行為模式。但當然這種事總是知易行難，只能慢慢練習XD

這張圖在說什麼？ 壓力可能一路推動焦慮、高度警覺、延宕與負面自我信念，最後讓我們更難照顧自己的基本需要。
今天的導讀主線 先理解壓力如何卡住，再看焦慮如何變成套路，接著回到大腦網絡、微創傷與新路徑練習。
Reading Map

導讀地圖｜從自動反應到重新選擇

為什麼我們會卡在熟悉的反應裡？

我們不只被外在世界塑造，也會被自己反覆相信的故事塑造。當「我不夠好」「我不能出錯」「我就是會想太多」這些敘事被重複太多次，它們會慢慢變成大腦熟悉的反應路徑，在我們還沒意識到之前先啟動。

重塑大腦，不是重新變成另一個人，而是在熟悉的反應之外，多練出一條可以選擇的路。
Before 我們被敘事塑造

外在環境會影響我們，反覆出現的自我敘事也會影響我們如何看待自己。

Pattern 重複會變成自動反應

大腦會優先使用熟悉路徑。負面想法被重複太多次，就會在意識之前先啟動。

Rewire 可塑性讓改變有路徑

舊路徑可以被強化，也可以被削弱。新的思想、習慣與行為，需要透過注意力、刻意練習與身體支持慢慢建立。

這本書的三段改變路徑

從看懂負面情緒如何形成，到改變內在敘事，再到支援大腦硬體，讓改變能持續。

03
維持與增強

增強積極性

照顧大腦硬體，讓睡眠、運動、休息與界線支撐長期改變。

Stress is not the enemy

壓力狀態：從短期動員到長期低電量

壓力不只有一種樣子。適度壓力能提升專注與反應，急性壓力讓身體短期動員；真正消耗人的，是長期無法恢復的慢性壓力。

導讀定位：看懂大腦何時從動員進入耗損

警覺性壓力

讓身體準備好面對任務，提升專注力與反應速度。這是一種可被使用的動員狀態。

急性壓力

短期能量爆發，協助處理眼前明確任務或問題。事件過後，身體需要回到恢復狀態。

慢性壓力

恢復不足，大腦優先維持基本功能，較難反思、學習與建立新習慣。

壓力狀態軸
警覺性壓力 專注、準備、可使用
急性壓力 短期能量、解決問題
慢性壓力 恢復不足、低電量模式
Low Battery Mode

低電量模式：大腦把資源留給生存

當大腦長期覺得自己在威脅中，它會把資源用來維持安全，而不是用來反思與改變。這也是壓力大時，我們更容易回到舊習慣的原因。

額葉皮質

像決策與判斷的控制中心。慢性壓力會削弱它調節情緒警報的能力。

杏仁核

像威脅偵測器。長期壓力會讓它更活躍，對壓力源更容易反應過度。

海馬體

與記憶、學習和關閉壓力反應有關。壓力過高時，整理經驗會變得困難。

Stress Response Curves

壓力曲線：關鍵在恢復能力

急性壓力像一波高峰，身體能在事件後回到尋常狀態；慢性壓力則讓交感神經長時間維持高張，副交感神經難以把系統帶回穩定。

導讀定位：用圖像說明恢復節奏
交感神經作用：警覺、動員、備戰
副交感神經作用：恢復、安定、回到基準

急性壓力：上升後能回復

壓力源出現後，交感神經快速動員；事件過後，副交感神經接手，系統逐漸回到穩定。

讀書會解讀：事件結束後能不能恢復，會影響大腦是否還有空間學習新反應。

慢性壓力：高張後難以回復

壓力源持續存在時，交感神經無法真正降下來，副交感神經的恢復力也被削弱。

讀書會解讀：慢性壓力會讓大腦更偏向生存反應，也更容易回到舊路徑。
Regulation Toolbox

恢復資源：先讓系統有電

建立新路徑需要大腦有可用資源。睡眠、身體、休息、界線與環境，都是改變能不能持續的條件。

導讀定位：從理解壓力轉向恢復資源
快速回到身體：生理性嘆息

連續兩次吸氣，再延長吐氣。適合在焦慮、緊張或情緒高張時使用。

走到戶外，看向遠方

開闊視野與自然環境能降低壓迫感，讓邏輯腦有時間重新介入。

更多恢復資源
睡眠、規律運動、冥想、自我催眠、接觸大自然、嗜好、減少不必要負擔、有限攝取咖啡因、學習說「不」而不內疚。
有目的地讓大腦隔絕刺激，適合用在會議空檔、重要通話前、演講前或緊張事件後。
運動、呼吸練習、完成小目標與穩定休息，能幫助大腦形成更有支撐性的回饋路徑。
Name the State

壓力、擔憂、焦慮：三種狀態，三種照顧方式

先判斷自己現在比較像任務壓力、未來擔憂，還是身體警報，才不會用同一種方法處理所有不舒服。

導讀定位：從壓力知識進入自我覺察
External demand

壓力

眼前真的有一件事需要處理，身體正在回應外在要求或威脅。

區辨問題 現在有一個明確的外在壓力源嗎？
對應照顧方式 把任務拆小，確認期限、優先順序、可用資源，並看見是否需要求助或減少負荷。
壓力需要可處理的任務結構。
Future thinking

擔憂

大腦正在模擬未來壞結果，常以「萬一……怎麼辦？」的形式出現。

區辨問題 我腦中是不是一直在想某個未來的壞結果？
對應照顧方式 把已經發生的事與腦中預想的壞結果分開，再找下一個可行的小步驟。
擔憂需要被整理成可檢查的句子。
Alarm system

焦慮

警報系統被啟動，眼前不一定有明確危險，身體卻已經進入備戰。

區辨問題 即使沒有明確事情發生，我還是感到害怕、緊繃或不安嗎？
對應照顧方式 先回到身體，例如生理性嘆息、看向遠方、放慢動作。等警報降低後，再思考問題。
焦慮需要先降低警報。

Habit Loop

焦慮迴路：警報、套路、短暫安撫

每次警報響起，如果我們都用反芻、逃避或短暫安撫處理，大腦就會更熟悉這條路。熟悉感會讓它下次更快啟動。

看見提示

最近是什麼事件、訊息、關係或念頭，最容易讓你進入警報狀態？

辨識套路

你通常怎麼讓自己暫時舒服一點？反覆思考、討拍、逃避、滑手機，還是讓自己更忙？

重新選擇獎勵

真正恢復的獎勵，會讓你回到身體與現實，而不是讓舊迴路更強。

暫停自動劇情

焦慮迴路啟動時，大腦很容易把情緒、推論與事實混在一起。這裡先做一個停頓：我現在相信的是事實，還是大腦補上的劇情？

情緒是真的 劇情需要檢查 事實與擔憂分開
Brain Networks

三個大腦網絡：內在故事與外在任務的切換

前面談的是生活經驗，這一段把它放回大腦網絡來看：為什麼我們有時會卡在內在反芻，有時又會用忙碌逃開自己的想法。

導讀定位：用大腦網絡統整自動反應

DMN：內在敘事與反芻

與神遊、創造性、自我指涉思考、自傳式記憶有關。當它過度活躍時，可能讓負面自我故事反覆播放。

CEN：專注、判斷與解決問題

負責外部任務、注意力、決策與問題解決。壓力太高時，這個網絡比較難穩定介入。

SN：注意力的轉接站

協助在內在思考與外在任務之間切換，幫助大腦判斷刺激的重要性。

當 DMN 開始反芻

01 看見念頭

這是一個事實，還是一個正在重播的解釋？

02 外部化反芻

把腦中的劇情寫下來，讓背景音變成可檢查的句子。

03 切換狀態

走動、看向遠方、接觸自然，先讓警報降一點。

Thoughts Become Beliefs

反覆走過的思想，會慢慢變成信念

從壓力、焦慮迴路到大腦網絡，我們看見舊路徑如何變得熟悉。接下來，導讀會往下看：這些信念如何被微小經驗累積、被大腦篩選現實，並透過練習慢慢改寫。

導讀定位：承接到信念形成與改寫
言語我怎麼描述自己與事件
思想我腦中重複播放的解釋
感受身體與情緒如何回應
行為我因此選擇或逃避什麼
信念我逐漸相信自己是誰
Micro Wounds

微創傷：小訊號長期累積，改寫自我感

有些影響我們很深的經驗，不一定是巨大而明顯的事件。反覆批評、否定、比較，或長期感到不被重視，都可能被大腦記成「需要保護自己」的訊號。

我不夠好 我不能出錯 我必須證明自己 我需要討好別人 我不值得被喜歡
核心觀念

焦慮、討好、完美主義或自我懷疑，有時是大腦在過去環境中學會的安全策略。

一次批評：當下難受，但很快被壓下去。
反覆比較：大腦開始追蹤自己哪裡不夠好。
長期否定：自我懷疑變成預設反應。
關係裡不被重視：對忽略、冷淡、延遲特別敏感。
大腦學到的保護模式 很多我們以為是個性的反應，其實可能是大腦長期記住重複訊號後形成的保護模式。
Belief Filter

信念濾鏡：大腦會搜尋熟悉的證據

大腦會用過去經驗建立記憶模板。這讓生活更有效率，也讓我們容易把新事件套進舊故事。真正影響情緒的，往往包含事件本身，以及我們如何解讀事件。

信念濾鏡
Belief Filter
朋友回訊息慢 → 我是不是不重要？
聽見一句批評 → 我果然不夠好
對方語氣冷淡 → 他是不是要離開我？

確認偏誤：舊信念會挑選證據

當心裡相信「我不受歡迎」，大腦會特別注意別人沒有回訊息、沒有主動邀請、沒有看見自己的時刻。久而久之，世界看起來就越來越像舊信念說的那樣。

事件 朋友訊息回得比較慢。
舊信念啟動 「我是不是不重要？」
情緒與行為 焦慮、失落、檢查手機、腦補、冷淡回應。
強化信念 關係變緊張，大腦更相信「我果然不被重視」。
New Path Practice

改寫路徑：四個可以練習的轉彎點

大腦會強化重複出現的路徑。當我們開始在負面情緒出現時練習新的反應，大腦也會慢慢熟悉新的選擇。

1

看見觸發

負面情緒通常被某些提示啟動。先看見提示，舊路徑就不再完全自動。

環境 某個人 社群媒體 睡前
2

承認舊路

可以對自己說：「我現在正在反芻。」「大腦正在走熟悉的路。」

關鍵 改變需要注意力與刻意。
3

穩定身體

情緒很強時，理性思考會變弱。先讓神經系統降下來，再整理想法。

深呼吸 生理性嘆息 走動 寫下念頭
4

正向強化

看見自己比以前更早覺察、更快停下來，就是在幫大腦記住新路徑。

可以改成 我這次有比以前更快發現自己在反芻。
新路徑靠一次次體驗被鋪出來。

「我怎麼又這樣？」會把注意力帶回自責；「我剛剛有停一下」會讓大腦記住新的選擇。神經可塑性不是瞬間換掉舊反應，而是讓新反應慢慢變熟。

Loss Adaptation

失去之後：大腦重建生活地圖

當我們失去某個人、一段關係、一份工作、一個夢想，甚至一種原本以為會延續的生活版本，大腦原本熟悉的神經路徑會被迫中斷。

悲傷是一段重新適應的過程。記憶系統、情緒系統、壓力系統和獎賞系統，都在學習如何在新的現實裡生活。

原本熟悉的生活預測
  • 有好消息想告訴某個人
  • 下班後習慣打給某個人
  • 每天為某段生活努力
  • 以為自己會走向某個未來

Closing Note

重塑大腦，是讓未來多一條路

舊路徑曾經可能保護過我們，所以改變不需要從責備開始。今天的導讀想留下的是：當壓力、擔憂、焦慮、失去或舊信念出現時，我們可以更早看見自己正在走哪條路，並開始練習下一個比較照顧自己的選擇。
